System Requirements

Scholar Navis utilizes local Embedding and Reranker models for maximum privacy and precision.

Minimum Configuration

  • Memory (RAM) 8 GB
  • Graphics (GPU) Not Required
    (CPU-only Inference)
  • Disk Space ~2 GB (Lite Models)
  • Models Arctic-XS / BGE-Base

Recommended (Pro)

  • Memory (RAM) 16 GB - 32 GB
  • Graphics (VRAM) NVIDIA 6GB+
    (RTX 3060/4060+)
  • Disk Space ~10 GB (High Precision)
  • Models BGE-M3 / Qwen2-7B-Embed

Note: High-performance Reranker models (like Gemma-9B) may require 20GB+ VRAM for optimal speed.
