Key Features

Built to streamline literature review, data analysis, and academic writing.

AI-Powered Insights

Seamlessly interact with your literature using integrated Large Language Models and Retrieval-Augmented Generation (RAG).

Modular Plugin Architecture

Highly extensible base tool classes. Easily add new analysis modules or custom workflows as independent plugins.

Local Vector Database

Securely index and retrieve your academic papers locally using ChromaDB, ensuring complete privacy of your research data.

System Requirements

Ensure your hardware meets the following criteria for optimal performance.

Minimum Requirements

  • OS: Windows 10/11 or modern Linux distribution
  • Processor: Intel Core i5 / AMD Ryzen 5 or equivalent
  • Memory: 8 GB RAM
  • Storage: 2 GB available space (SSD recommended)
Recommended

Optimal Configuration

  • OS: Linux environment (or Windows with WSL2 configured)
  • Processor: Modern multi-core CPU
  • Memory: 16 GB to 32 GB RAM (for large dataset processing)
  • Graphics: NVIDIA GPU (for accelerated local LLM/embedding inference)
